# 🛡️ Fire & Incident Dashboard

A professional real-time monitoring system for emergency services (Fire, Traffic, Floods).

## ✨ Key Features
- **Smooth Auto-Zoom:** Click an incident in the sidebar to fly to its map coordinates.
- **Pulse Markers:** Emergency locations feature animated pulsing markers for high visibility.
- **Glassmorphism Sidebar:** Modern semi-transparent UI that floats above the map.
- **Theme Engine:** Full Light/Dark mode support with persistent storage.
- **Incident Management:** Add incidents via coordinates with a sleek dropdown menu.

## 🚀 Setup
1. Install requirements: `pip install flask flask_sqlalchemy folium`
2. Run `python app.py`
3. Navigate to `http://127.0.0.1:5000`

## 📂 Project Info
- **Frontend:** HTML5, CSS3, JS (Leaflet/Folium)
- **Backend:** Python (Flask)
- **Database:** SQLite
